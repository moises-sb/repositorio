Practica de Git

Comando Descripcion
git init Inicializa la carpeta como un repositorio de GUT para empezar a gestionaar el cambio
git add -A Carga los cambios realizados en el proyecto
git add status Verifica los cambios realizados en los commit
git commit Pasa los archivos del area de stagging al area de repositorio, es necesario poner un mensaje que represente todos los cambios del proyecto
git push origin master Manda los archivos del area de repositorio a un servidor por ejemplo github, bitbucket, gitlab, etc.
git log Muestra todos los commit realizados
